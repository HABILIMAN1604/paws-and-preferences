# Paws & Preferences
A premium cat discovery experience built for **Netizen Experience**.

## UX Goals
* **Smooth Interaction:** Use swipe gestures with Framer Motion so moving between cats feels natural.
* **Micro-Interactions:** Small animations when a user likes or dislikes a cat to make the app feel more interactive and fun.
* **Mobile Friendly:** Designed mainly for phones so users can swipe easily with one hand.

## Tech Stack
* **React + Vite + TypeScript:** For a type-safe, high-performance foundation.
* **Tailwind CSS:** For rapid, consistent UI development.
* **Framer Motion:** For gesture-based animations.
